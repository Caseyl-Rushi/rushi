# 如是 · 游戏设计文档
## 第五章：街区与城市空间

> **核心原则**：天空之城是我们的梦，地面都市是我们必须面对的现实。城市的布局不是规划出来的，是长出来的。公共空间大于私有空间——这个世界相信"共同"比"独自"更重要。阳台的视野是你所有努力的结算画面——你站在云端，往下看。

---

### 零、世界的双层结构：云端与地面

如是的世界由两层空间构成：

```
┌─────────────────────────────────────┐
│          ☁️ 天空之城（初始）           │  ← 梦境 · 庇护所 · 起点
│    云端浮岛 · 斜 45° 等距视角         │
│    节奏慢 · 温暖 · 不焦虑             │
│                                     │
│    ┌──阳台 ──→ 往下看 ──→ 地面       │
│    │            ↓                    │
│    │     ☁️☁️☁️☁️☁️☁️               │
│    │     ☁️☁️☁️☁️☁️☁️              │
│    │     ════════════               │
│    │                              │
│    │   🏙️ 地面都市（进阶）           │  ← 现实 · 问题堆积处 · 终局
│    │   摩登城市 · 复杂决策 · 冲突      │
│    │   文明走向的决定在这里发生        │
│    └──────────────────────────────┘
└─────────────────────────────────────┘
```

**天空之城 = 梦。** 这里节奏慢，阿琳的面包店不慌不忙，长椅上的人可以空坐一个下午。资源有限——因为没有土地，你得在浮岛上精打细算。但这里不焦虑。这里是极乐世界，云端之上的一片净土。

**地面都市 = 实践。** 你要下去。梦是起点，不是终点。地面才是问题堆积的地方——更多的街区、更复杂的决策、更大的冲突。下去的人，才能真正影响文明的走向。

**从梦到现实的旅程不是传送，是降落。**

#### 天空之城与地面的分工哲学

天空之城承载的是**生活的边边角角**——一只猫、一条围巾、面包店的酵母、老树下的铁盒、张叔没修好的那把茶壶。这些事不宏大，但它们是一个人在宏大世界里仍然能保持柔软的底线。

地面都市承载的是**文明的宏大选择**——太空港选址、能源站建设、跨区宪法修订、与外星文明第一次对话的语气。这些事决定了 300 年后人类站在终点时的姿态。

去地面的人在做决定。留在天空的人在被决定影响。

但有一天，地面回来的人坐在云端长椅上往下看。他看到的不是"一座城市"。他看到了那条街上有一个卖花的小姑娘。他突然想知道那把茶壶修好了没有。

一个人在议事厅投了一天的票、吵了几场架、做了一个影响千万人生活的决定——他回到云端，在老树下坐下，看着自己的皮影被傍晚的光拉长。他需要这一刻。不是因为累了，是因为他需要被提醒自己为什么做了那些决定。他在地面上造的太空港，是为了让云端这个在长椅上发了一下午呆的学生有一个未来。

**天空之城不是新手区。天空之城是归处。地面是去处。**

### 一、斜 45° 空间结构：自然生长，不画格子

#### 布局原则

街区不是棋盘。建筑沿着地势和道路自然分布——有的聚在一起，有的散开。一条街可能弯一下，因为这里有棵老树。几栋楼可能挤在一起，因为那时候第一批居民舍不得离得太远。

**设计规则**（不是视觉规则，是生成规则）：

| 规则 | 说明 |
|------|------|
| 沿路生长 | 建筑优先沿街道两侧排布，但不强制等距 |
| 地标引力 | 重要公共建筑（议事厅、市集、老树）周围建筑密度自然更高 |
| 地形顺从 | 如果有坡地、河岸、树林，建筑会绕开或嵌入，而不是铲平 |
| 簇群间隔 | 密集的建筑簇群之间，自然留出空地——可能是小广场、菜园、或什么都没规划的野地 |

**视觉参考**：欧洲中世纪小镇、日本京都的町屋街巷、中国南方水乡的沿河民居——不是照搬建筑风格，是照搬**"人住久了，路自然就有了"的逻辑**。

#### 地块系统

每块可经营的用地是一个**格位**。格位不要求对齐、不要求正方。大小可以不同——靠街的窄一些，里面的宽一些。公园边上的可能有树荫加成。

斜 45° 等距视角下，格位表现为一个菱形（等距投影下的矩形地面）。建筑立在这个菱形上。

#### 天空之城的特殊视觉

- **脚下不是土，是云层。** 半透明的云雾在街区边缘流动。走到街区边缘，能看到脚下的虚空和远处的地面轮廓。
- **栏杆。** 街区外围有护栏/矮墙——既是安全边界，也是视觉提示："这里是边缘。"
- **云层动态。** 云会缓慢流动、聚散。天气变化在云层上表现得更明显——晴天云薄如纱，雨天云厚如幕。
- **向下看。** 在天空之城任何位置，视线向下都能隐约看到地面的城市剪影——被云雾半掩着，像水中的倒影。

### 二、天空之城（初始阶段）：从一棵树开始

#### 天空之城的初始结构

天空之城围绕一个**中心地标**展开。它是一座悬浮在云端的浮岛城镇。初始只有 3 个街区，每个街区约 30-50 栋建筑（含住宅、店铺、公共设施）。

100 栋建筑是否太多？取决于"建筑"的定义：

| 类型 | 占比 | 说明 |
|------|------|------|
| 居民住宅 | 归景观范畴 | 玩家住所 + NPC 住所。一栋楼可容纳多个住户。住宅楼本身是视觉元素，不参与公私比例计算 |
| 经营店铺 | 25% | 玩家的店、NPC 的店（面包店、杂货铺等） |
| 公共建筑 | 25% | 议事厅、市集、图书馆（待建）、诊所、公告板、公园、老树长椅 |
| 空地/预留 | 10% | 未开发地块。未来提案可以发挥作用的空间 |
| 开放空间 | 40% | 公园、广场、步道、观景台、云边露台。这是天空之城最珍贵的部分——站在云端看世界的地方 |

**公域私域比 = 5:5**（公共区域占 50%）。天空之城的核心体验不在个人家里，在街上、在议事厅、在公园长椅上、在云边的观景台上。

#### 天空之城固定公共设施

| 设施 | 功能 | 位置 |
|------|------|------|
| 议事厅 | 投票、讨论、历史档案 | 街区中心 |
| 公告板 | 信息流、悬赏、失物招领 | 街角/议事厅旁 |
| 市集 | 交易场所、旅行商人落点 | 两街区交汇处 |
| 公园 + 老树长椅 | 休息、偶遇、对话 | 每个街区 1 个 |
| 诊所 | 居民健康（NPC 陈医生） | 街区边缘 |
| 小图书馆（待建） | 知识积累、文化指数 | 空地——第一个悬赏的产出 |
| 杂货铺 | 基础物资 | 街角 |
| 阿琳的面包店 | 食物 + 社区八卦 | 街角 |
| **降落平台** | **通往地面都市的唯一通道** | **天空之城中心** |
| **云边观景台** | **往下看地面城市的最佳位置** | **每个街区 1 个** |

#### 天空之城的三个初始街区

| 街区 | 气质标签 | 地标 | 初始 NPC |
|------|---------|------|---------|
| 第 7 街区 | 混合型（新居民默认入住） | 老树 + 公园长椅 | 阿琳、林教授、陈医生 |
| 第 3 街区 | 偏文化/安静 | 小图书馆选址地 | 待定 |
| 第 5 街区 | 偏商业/热闹 | 市集 | 待定 |

第 7 街区是所有新玩家的默认起点。不是因为最好，是因为最"普通"——一个混合街区，什么都有，什么都不极端。玩家可以在入住后申请迁移到其他街区。

**无论文明迭代到第几轮，天空之城永远存在。** 地面上的城市可能已经运转到第 6 轮、建了 12 座图书馆、经历过两场内战——但云端永远是新人的入口。新人从云端开始，向下看到的是一个**已经有历史的世界**。

### 三、NPC 的范式作用

NPC 不只是背景装饰。他们的行为向玩家**演示这个世界怎么运转**，但不通过对话框教学。

NPC 的职业应该丰富多样——他们演示的是：在这个世界里，你可以悠闲地只经营自己的小世界，也可以参与更大的事物，甚至最终决定某些事物的走向。

| NPC | 职业 | 日常行为（玩家可观察） | 范式作用 |
|-----|------|---------------------|---------|
| 阿琳 | 面包店主 | 每天清晨第一个开店。菜单每天更换 1-2 项。会跟常客聊社区八卦。偶尔在公告板留言 | **经营示范**：开店、换菜单、与邻里互动——新玩家看她怎么做，自然学会经营 |
| 林教授 | 退休学者 | 常在公园长椅看书。偶尔在议事厅发言。会在公告板发"寻人一起整理旧书" | **公共参与示范**：提案、讨论、寻找合作——演示如何发起和参与公共事务 |
| 陈医生 | 诊所医生 | 每天在诊所。偶尔在公园散步。遇到紧急事件会主动去议事厅 | **社会责任示范**：专业技能可以服务社区 |
| 杂货铺老板 | 商人 | 每天开门。偶尔进新货。会和旅行商人讨价还价 | **交易示范**：物资流动、定价、议价 |
| 旅行商人 | 流动 NPC | 不定期出现在市集，带来外面的商品和信息 | **外部世界触达**：提醒玩家——小镇外面有更大的世界 |
| **老周** | **花匠** | **每天在云边照料花草。偶尔把新品种的花送给邻居。会在观景台坐很久看下面** | **生活美学示范**：你可以只做一件事，把它做到极致，这也是一种活法** |
| **小叶** | **学生/学徒** | **经常去图书馆旧址翻书。向林教授请教问题。偶尔在公告板上发一些天真的想法** | **成长路径示范**：新手如何从一个观察者变成参与者** |
| **张叔** | **修理匠** | **店铺在街区角落。什么都能修。经常帮邻居免费修小东西。话不多但大家都认识他** | **利他行为示范**：不求回报的服务也是贡献的一种** |
| **苏姐** | **茶馆老板娘** | **茶馆傍晚开门。人们下班后聚在这里聊天。她记得每个人的口味** | **社交枢纽示范**：一个地方让人们在非正式场合自然相遇** |
| **老钟** | **前地面都市居民** | **偶尔从地面回到天空之城探亲。会给天空之城的人讲下面的故事。他的话里带着一种过来人的语气** | **过渡桥梁示范**：连接两个世界的人，暗示"你也可以下去"** |

NPC 不会弹窗说"你可以去公告板看看"。但新玩家在街上走一圈，看到阿琳在公告板前停了几秒、林教授坐在长椅上写东西、老周在云边看风景、老钟在茶馆里讲下面的故事——他会自己理解：这些人都在做自己的事。我也可以。

### 三-二、皮影人：这个世界里的每一个人

#### 视觉设定

所有角色——玩家、NPC、每一个在街上走动的人——都是**皮影质地的人形**。不是像素，不是日式立绘，不是 3D 模型。是皮影。

**正面**：一个人的简笔轮廓。年龄、职业、气质可以用线条的粗细和节奏分辨——阿琳的线条圆润温暖，林教授的线条瘦长斯文，老张的线条粗粝结实。没有五官细节，但你知道那是谁。

**侧面**：极薄。从斜 45° 看过去，是一张皮立在街道上。新人几乎是一条线，老居民有了一点厚度。

**背面**：和正面不同——不是空的。背面写满了字。不是系统标注的属性，是这个人**做过的事**。日程本上"昨日的踪影"全刻在背面。小字，密，沉默。但存在过。

#### 皮影的动

皮影有**关节**。头、身、臂、腿由隐形丝线连接。走路时袖袍轻摆，坐下时身体微倾。不需要 3D 骨骼，几个关节点 + 2-3 帧动画就能让皮影活起来。

皮影有**光源**。天空之城漂浮在云上，阳光从不同角度打下来。正午影子短——皮影人脚下一个小小的黑色菱形。傍晚影子长——皮影被拉成一条长长的墨线，拖在街上。下雨天影子模糊，雪天皮影上会落白。

你可以什么都不做，走在街上，看着自己的影子。

#### 纸的厚度

一张皮影的厚度不是固定值。它是**时间的计量**。

- 一个贡献值 0 的新人——皮很薄。侧面几乎是一条线。
- 一个在这里生活了 200 年的老居民——皮厚了。侧面有真正的轮廓。
- 一个离开的人——皮影留在街上。正面褪色，背面已经写满。他已经不在了，但他的皮还立在那里。走近它，你能看到上面的字："帮邻居找了猫""在第六次空地方案的投票中投了图书馆"。

厚度是一种**无声的身份识别**。你走在街上，不需要看任何 UI，就能分辨一个老居民和新来的人。不是靠标签，是靠体量。新人不会因为"薄"而被轻视——每个人都从这里开始。但你路过老张的时候，你意识到这个人在这里站了很久。

#### 来的时候是一张纸，走的时候是一本书

一个新玩家进入如是——他是全新的皮影。身上是白的，背面没有字。什么都没发生过。

他在这个世界里生活了 300 年——一轮文明。面包店开了 30 年。在议事厅投了 47 次票。在公园长椅上坐过了四季。帮邻居找到了猫。

当这一轮文明结束——他的皮影已经写满了。正面简笔还在，背面是小字密麻的一生。

新的一轮文明。另一个玩家住进这条街。他路过老树下。树旁边立着一个旧皮影——那是上一轮的人留下的。正面已经看不清脸了，只有轮廓。背面全是字。

他站了一会儿。然后继续走。

#### 技术优势

皮影人是极低成本的 2D 角色方案：
- 不需要 3D 建模
- 不需要骨骼动画——关节摆动 2-3 帧即可
- 斜 45° 视角下，正面 + 侧面薄边 = 天然立体感
- 美术风格统一——中国皮影/剪纸传统，参考资源极丰富
- 厚度变化 = 简单的 scaleX 或侧面宽度参数，一条代码的事

### 三-三、人味时刻

人味不在系统设计里，在系统**留出的空间**里。以下不是任务、不是事件、不是系统触发的内容。它们是**可能发生的事**——不需要任何人批准，自然会发生在有人的地方。

#### 告别的仪式

阿琳的面包店开了 30 年。某一天，她的公告板留言变成：

> "今天的面团揉完了。明天开始，这个面包店由新的人接手。谢谢 30 年来的每一只牛角包。"

这不是系统事件。这是她真的离开了。第二天，新店主上任。老顾客去店里，面包还是那个味道——但菜单上多了一行小字："阿琳的配方。"

她去哪了？没人知道。但她那张旧皮影还在面包店角落里——正面是一个模糊的影子，背面全是字。

#### 地下的信

第一轮文明时有人在老树下埋了一个铁盒，里面是一封信。第二轮时有人挖到了它。信上写着：

> "我不知道 300 年后还有没有人住在这里。如果有人读到这行字——你好。这棵老树是我种的。希望它还活着。"

读信的人可能不在这条街上了。但铁盒被放回了原处。上面多了一张新纸："它还活着。"

#### 无用的回报

你昨天在公告板上帮人找到了猫。今天你发现你的门把手上挂了一袋橘子。没有署名。你翻开日程本——"你昨日的踪影"里多了一行：

> "有人在你的门把手上挂了一袋橘子。可能是那个找猫的人。"

你做了事。你没有要求回报。但回报自己找上门了。你不需要知道是谁。这行字就够了。

#### 张叔的下午

有一天你登录，发现修理摊的张叔坐在长椅上，没在店里。他不是在等人。他昨天没修好茶馆的一把茶壶，他在跟自己较劲。

你可以走过去。在旁边坐一下。你可以说"没事，茶壶我帮你看看"。你也可以什么都不说。

过了一会儿，他站起来。回店里。门开了。

这不是任务。这是一个上了年纪的修理匠和一把没修好的茶壶。但你在旁边坐了一下。

#### 边边角角的社会逻辑

有些事本身不说话，但玩家会自己想：

- 公告板上有人贴"愿意帮老人代买杂货"——老龄化、独居。但你没看到有人在讨论老龄化。你只看到一个人在做没人要求的事。
- 旅行商人带来的某种商品引发了排队——稀缺性、从众心理。你排了吗？没排的话你确实不需要那东西。
- 老钟说"下面的人不看你了"——阶层分化。你还不确定你要不要下去。
- 林教授的学生申请了一个提案，被多人否决——代际参与度不平等。你觉得不公平，但你不知道怎么投那关键的一票。
- 面包店原料涨价了——阿琳没声张，但菜单薄了。你注意到了，多买了一包。你是她的老顾客。

这些事件不贴标签。一个玩家可以连续一个月只买面包、看风景。他永远不会被推到这些社会逻辑面前。但如果他停下来多想一秒——他想了。

#### 人味时刻的设计原则

人味时刻不是"加更多内容"。是**确保系统不会把这些空间挤掉**：
- 不给 NPC 行为加过多的约束规则。阿琳想离开就可以离开
- 让物品可以在世界中被"留下"（铁盒、信、橘子）
- 日程本记录的不是数值变化，是**连续性的证明**——"有人在你的门把手上挂了一袋橘子"
- 倒计时不催促这些微小的事——陪你熬过 300 年的不是大事件，是每天琐碎的近旁

### 四、从天空之城到地面都市：降落

#### 降落的触发

不是等级到了自动传送。不是系统说"恭喜你解锁大城市"。

是**你自己的创造欲超出了天空之城的资源边界**。

具体触发条件（玩家主动选择）：

| 条件 | 说明 |
|------|------|
| 经营等级达到上限 | 你的店铺在天空之城已经升到头了——这里的客流量、材料供应已经满足不了你的下一个想法 |
| 公共参与深度达到 | 你的贡献值达到一定级别（如 500+），在天空之城的提案对你来说已经不够复杂 |
| 有跨街区/城市级提案 | 你提交了一个需要地面都市资源才能实现的方案——天空之城没有这种材料、技术或人才 |
| 个人选择 | 纯粹是"我想去更大的地方"。不需要理由 |

#### 降落的方式

不是"点传送"。是一段**有仪式感的旅程**。

1. 玩家走进天空之城中心的**降落平台**
2. 平台上有电梯/飞艇/某种连接天空和地面的垂直通道
3. 画面切换为下行视角——云从窗外掠过，地面都市逐渐清晰
4. 你落地。第一次落地时，这是一个仪式。之后往返，就是日常

**核心体验**：你在告别什么，也在面对什么。

#### 降落后保留什么

| 保留 | 变化 |
|------|------|
| 贡献值 | 不变 |
| 通用货币 | 保留，但地面都市的物价更高 |
| 经营历史 | 你在天空之城的店保留（可以转给 NPC 或其他玩家经营），也可以直接关掉 |
| 人际关系 | 你在天空之城认识的人——他们的联系方式保留在 App 的"邻里"中。去了地面，你们还是认识 |
| Token 配额 | 不变 |
| **随时回访权** | **你可以随时坐电梯回到天空之城。这不是单程票。** |

#### 降落后的变化

- 更大的地图——更多的街区、更多的公共设施、更复杂的决策
- 委员会从 NPC 转为玩家选举（当第一批到达地面且贡献值 ≥ 500 的玩家 ≥ 5 人时触发首次选举）
- 新资源类型——天空之城没有的建材、技术、文化资源
- 新决策类型——城市规划、区域合作、跨区法规
- 远处的"地面都市"变成了你脚下真实的街道。窗外的新远处——可能是其他城市、可能是太空港、可能是文明级设施

#### 天空之城的角色变化

天空之城不因玩家离开而废弃。它继续运转——新玩家入住，新的提案产生，老树还在。离开的玩家可以通过 App 看到天空之城的动态。某天你可以"回去看看"——阿琳还在烤面包，公园长椅上换了新人，云还是那样缓缓流过。

**天空之城永远是庇护所。不是新手保护区。一个在地面混得灰头土脸的人，可以坐电梯回到云端，在阿琳的面包店坐一会儿，在老树的云边长椅上发一下午呆，然后第二天再下去。**

### 五、阳台视野：文明的结算画面

阳台是你所有努力的**视觉回报**。它是**云端阳台**——悬在云层边缘，脚下是流动的云雾，远方是若隐若现的地面都市。

#### 基础变化（天时）

| 维度 | 变化 |
|------|------|
| 季节 | 春夏秋冬——云的颜色、厚度、流动速度随之变化。春云轻薄，冬云厚重 |
| 天气 | 晴、雨、雾、雪。雨天云层加厚，能见度降低。雪天云边结霜。重大事件时可能有特殊天气（外星信号异常→云层出现异常光晕） |
| 昼夜 | 清晨云海泛金，正午云白如练，傍晚云染暖色，深夜星空穿透云隙。倒计时在每种光线下都有不同的"读感" |

#### 高度计

阳台屏幕旁边，除了倒计时，还有一样东西：**高度计**。

它不只是数字。它告诉你，你离地面还有多远。也许有一天，你往下看的时候，高度计显示的数字……在变小。因为你已经在往下了。

高度计是隐喻：**你离现实越近，离梦境越远。但你始终可以选择回来。**

#### 文明级变化（人和）

这是阳台视野最核心的设计：**你往下看到的地面都市轮廓，随着文明的整体进展而变化。**

| 文明阶段 | 阳台往下所见 |
|---------|------------|
| 天空之城阶段（初始） | 云雾深处，隐约可见地面都市的剪影——高楼林立，灯火通明。你看不清细节，只知道那里有一个巨大的东西在等你 |
| 初次降落期（第一批玩家下到地面后） | 地面都市轮廓清晰了一些。你能分辨出几个区域的灯光分布。你知道有人下去了 |
| 城市繁荣期（大量玩家在地面活动） | 地面都市的天际线清晰可见——议政塔、图书馆穹顶、市集广场的灯光。你甚至能看到降落平台的灯光在地面上亮起 |
| 行星级文明 | 地面都市的边界开始模糊。远处出现巨型建筑——能源站、生态穹顶。夜间有更多灯光向上延伸 |
| 恒星级文明 | 天空出现轨道电梯基座、近地空间站灯光、往来轨道的飞船尾迹穿过云层。倒计时屏幕上多了一行字——"外层空间监测站已激活" |
| 未知接触前夕（300 年终点） | 云层下方出现异常光晕。不管是什么——你在阳台上看了 300 年（如果一轮走到终局），最后的画面是那一刻的天空 |

**核心体验**：站在云端阳台上，你看到的不只是风景。你看到的是**你和其他人一起建了 300 年的东西**。每一盏灯后面，都是一个和你一样——每天站在这里、翻开日程本、然后走下楼的人。

### 六、空间与技术实现的衔接

#### 斜 45° 渲染的基础方案

- 地面层：菱形瓦片拼成的等距网格（或使用自由摆放坐标的 Sprite 层）。天空之城的地面材质是半透明云纹而非泥土
- 云层渲染：Canvas 多层半透明渐变 + 缓慢位移动画。云层在街区边缘更浓密，中心较稀薄
- 建筑层：每个建筑的正面和侧面两张图（或使用 2.5D 的简单建模导出）
- 遮挡处理：Y 轴越靠前（越靠近屏幕下方）的对象渲染层级越高——这是等距视角的标准做法
- 边缘效果：街区边缘有护栏/矮墙模型。走到边缘时，镜头可以轻微倾斜，展示"往下看"的视角
- 地块系统：每个格位用坐标标记（x, y），建筑的可放置/不可放置由 tag 控制

#### 有机布局的技术考虑

"不画格子"是设计原则，技术上需要处理：
- 格位不要求整齐排列——相邻格位可以偏移、错位
- 道路不是直线，是贝塞尔曲线或折线
- 建筑在格位上可以微调位置（不用严格居中）
- 云层的深度层次：前景云（遮挡部分视野）+ 中景云（街区地面）+ 背景云（地面都市远景）

实现难度不高——HTML5 Canvas 完全能做。只是比严格格子的等距地图多写一些坐标偏移逻辑和云层渲染。
