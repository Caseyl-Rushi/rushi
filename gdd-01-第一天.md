# 如是 · 游戏设计文档
## 第一章：第一天

> **核心原则**：把现实中缺的、现实中想要的，用虚拟空间尽可能找到。整个过程干净、纯粹、舒服。
>
> **设计约束**：不催促、不弹红点、不塞任务、不替玩家做选择。一切交互通过触碰和移动完成，不使用对话选择框。阳台是所有循环的起点和终点。

---

### 世界观速览

300 年后，一种未知的外星文明将抵达地球。其意图不明，但人类必须做好两手准备。地球上最后的城市正在运转，你身处其中一个外围小镇。远处的大城市灯火通明——那是这里的核心，也是你终将走向的地方。

但今天，你只是刚刚落地。你有一小块地，一间住所，和一本日程本。

倒计时从你进入这个世界的第一秒开始：109,575 天。

---

### 场景一：阳台（登录）

**视觉**：你站在一栋居民楼的阳台上。时间是傍晚。城市在你面前展开——远处是高楼林立的城市核心区，灯火密集。近处是你所在的小镇，街道安静，矮房错落，几扇窗户亮着暖黄色的光。

**没有**：对话框、任务提示、高亮箭头、新手引导弹窗。

**有**：阳台栏杆上嵌着一块屏幕，显示倒计时数字：**109,575 天**。

玩家手边放着一本日程本。点击或触碰即可翻开。

---

### 场景二：日程本

翻开日程本，看到今日内容。页面设计干净，纸张质感。如下：

---

**今日**

**入住信息**

地块面积：临街空地 + 一间住所

---

**本周**
（翻页可看）

（翻页可看）

---

玩家可以：
- 合上日程本，回到阳台看风景
- 转身走进屋里（住所内部，待设计）
- 从阳台下楼，进入街区

所有操作无提示。日程本仅仅是信息载体。

### 场景三：街区街道（斜 45° 等距视角）

**视角切换**：从阳台下楼后，画面从横向卷轴自然过渡为斜 45° 等距视角。玩家的角色出现在街区街道上，站在 B-4 号地块前。

**B-4 号地块状态**：一小块临街空地，点缀着自然生长的杂草，但不显荒芜。左侧相邻是一家已开业的杂货铺，右侧是空地。对面是一棵老树，树下有一张长椅。

**屏幕上浮现一行淡色文字**：

> 这是你的地。你可以在这里开店、种花、或者什么都不做。

文字停留数秒后自行消失。不遮挡操作。

**可互动的建筑与元素**（当前街区视野范围内）：

| 位置 | 内容 | 交互方式 |
|------|------|---------|
| B-4 号地块 | 你的地 | 点击进入经营意向登记 |
| B-3 号 | 杂货铺（已开业） | 点击查看商品、与店主点头致意 |
| 对面 | 公园长椅（老树下） | 点击坐下或离开 |
| 街角 | 街区公告板 | 点击查看、留言 |
| 街道拐角 | 阿琳的面包店 | 点击进入 |
| 远处天际线 | 城市核心区剪影 | 不可到达，仅视觉锚点 |

---

### 场景四：经营意向登记（通过"街区通" App 或 B-4 地块触发）

打开"街区通" App 或在 B-4 地块上点击，进入经营意向登记界面。

**不是下拉菜单。是一张空白卡片。**

界面顶部一行字：

> 你打算在这里做什么？写下来。可以很短。（之后可以改）

卡片下方，是其他居民留下的真实文字示例——不是系统生成的占位符，而是街区内已有玩家或 NPC 的经营宣言：

> *"我想开一家小书店，专门放那些绝版的书。"*

> *"烤面包。一直想烤面包。"*

> *"还没想好。先住下来再说。"*

> *"修东西的店。什么都修。"*

玩家可以在空白区域输入自己的经营意向，一句话足够。也可以留空，系统不催促。

提交后，日程本自动更新：
```
经营意向：已登记
"（玩家填写的内容）"
```

如未提交，日程本保持"尚未登记"。无任何提醒或催促。

**经营意向不是职业锁死**：玩家随时可以在 App 中修改。你的店可以今天是个书店，下个月变成花店。这是你的空间。

---

### 场景五：走出门，认识这条街

玩家从 B-4 地块沿街行走。斜 45° 等距视角下，街道横向可左右滑动，但初始范围受限——当前可见的只有这一段街。

**路过杂货铺**：窗口有人探出头，对你点头。可能是 NPC，也可能是真实玩家。交互方式为点击窗口查看商品列表，或简单致意。

**走到公告板前**：点击查看完整内容。公告板是街区的信息流，内容由玩家提案、系统通知、NPC 日常消息混合组成。

当前公告板内容示例：

```
━━━━━━━━━━━━━━━━━━━━━━━━
📢 街区迎新
本月新入住居民：3 人
欢迎 B-4、B-7、C-2 的邻居

━━━━━━━━━━━━━━━━━━━━━━━━
🏗️ 本周悬赏
小镇西侧空地规划方案征集
奖励：200 贡献值
截止：本周五 20:00
[查看详情] [提交方案]

━━━━━━━━━━━━━━━━━━━━━━━━
🧣 失物招领
谁在公园长椅上落了一条蓝色围巾
请到杂货铺认领

━━━━━━━━━━━━━━━━━━━━━━━━
🍞 阿琳的面包店
今日剩余：全麦吐司 4 条、可颂 6 个
明日菜单预告：黑麦酸面包（新品）
```

**公告板不是 UI 菜单**。它是一块真实的板子，贴在街区街角。上面的每一条内容都是这个世界里真实发生的——有人丢了围巾，有悬赏在进行，阿琳的面包店今天还有货。

**走到公园长椅（老树对面）**：树下长椅上坐着一个人。可能是 NPC（如林教授），也可能是其他玩家。

- 点击长椅：坐下
- 坐下后，旁边的角色可能说一句话，也可能不说
- 说话内容由当天街区动态生成——"你看到公告板上那个围巾了吗"、"西区那块空地，你觉得建什么好"
- 再次点击：站起来走开

**没有"按 A 交谈 / 按 B 离开"。你只是坐下，或者走开。**

**走到阿琳的面包店**：点击进入。店内展示当日菜单和剩余数量。可以购买（使用通用货币）。阿琳可能简短互动——"你是 B-4 新来的？欢迎。"购买记录会留在你的日程本和她的顾客记录里。

### 场景六：傍晚回到阳台（退出前）

玩家在街上走了一段时间后，可以随时回到住所，走上阳台。

**阳台视野**：
- 近处：你的街区，路灯刚亮，街道安静。杂货铺关了门。公告板前有人在停步。
- 远处：城市核心区灯火通明。倒计时屏幕又跳了一格。

**翻开日程本**。内容已悄然更新——不是系统"推送"的，是你今天做过的事自动留下了痕迹：

```
今日记录
───
16:23  坐在公园长椅上
16:41  在公告板看到了周五悬赏
16:55  在阿琳的面包店买了 1 条全麦吐司
```

日程本不评判你的今天。它只是诚实记录。

明天是周一。翻到下一页：

```
明日（周一）
───
· 悬赏"西侧空地规划"提交截止：周五 20:00
· 公告板更新：预计有新提案
· 阿琳预告：黑麦酸面包（新品）
· 街区议事厅：本周议题将于明早发布
```

玩家：
- 可以合上日程本，最后看一眼远处的城市
- 倒计时又跳了一下
- 转身走进屋里——退出游戏，或者进入住所内部（住所功能待后续设计）

**退出不需要"保存"。一切自动记录。**

阳台，归来时仍是这里。离开时也是这里。

### 第一天结束时的玩家状态

| 项目 | 状态 |
|------|------|
| 经营意向 | 已登记（或未登记——尊重玩家选择） |
| 通用货币 | 初始赠送少量，可供购买基础商品 |
| 贡献值 | 0（尚未参与公共事务） |
| 投票权 | 1 票（初始） |
| 日程本 | 记录了今天的轨迹 |
| 邻里关系 | 可能认识了阿琳、杂货铺老板、长椅上的人 |
| 对世界的感知 | 知道有悬赏、有公告板、有倒计时、远处有个大城市 |

### 第一天设计检查清单

对照核心原则自查：

- [x] 没有任何对话框强制选择
- [x] 没有任何红色提示点或任务箭头
- [x] 玩家可以做任何事，也可以什么都不做
- [x] 信息呈现方式为日程本、公告板、App——玩家主动获取，非系统推送
- [x] "拥有一块自己的地"——B-4 号地块从第一天起就是你的
- [x] "牵挂"的种子已经播下——周五的悬赏、明天的菜单、那块空地建什么
- [x] 阳台作为起点和终点，承载登录和退出的全部情感重量
- [x] 远处的大城市作为视觉锚点，持续的轻度向往，非系统催促
- [x] 干净——界面元素克制，文字平淡但准确
